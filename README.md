🎮 VP Optimizer – Valorant

O VP Optimizer é uma ferramenta web que calcula a forma mais barata de comprar VP (Valorant Points) para adquirir skins no jogo Valorant.

Ele compara automaticamente diferentes opções de compra, incluindo:

Loja oficial
Revendedores (como RDC)
Uso de ticket de desconto da Riot
Cashback progressivo
🚀 Funcionalidades
💰 Cálculo automático do custo mínimo para atingir o VP desejado
🧠 Otimização inteligente combinando pacotes
🎟️ Suporte a Ticket Riot (-50 VP)
💳 Simulação de cashback progressivo (RDC)
📊 Comparação direta entre:
Loja Oficial
Rei dos Coins (RDC)
🏆 Destaque automático da opção mais barata
📉 Exibição da economia total
🖥️ Tecnologias utilizadas
HTML5
Tailwind CSS (via CDN)
React 18 (via CDN)
Babel (para JSX no navegador)

⚠️ Não é necessário instalar nada — roda direto no navegador.

📦 Como usar
Baixe ou copie o arquivo .html
Abra no navegador (Chrome, Edge, etc.)
Preencha:
VP atual
VP necessário (valor da skin)
Marque opções se desejar:
✔️ Ticket Riot
✔️ Cashback
Veja o resultado otimizado automaticamente
🧮 Como funciona o cálculo

O sistema:

Calcula o VP faltante
Gera combinações possíveis de pacotes
Avalia cada combinação com base em:
Menor custo total
Maior VP entregue (em caso de empate)
Menor quantidade de compras
Para RDC:
Aplica cashback sequencial (5%)
Usa saldo acumulado na próxima compra
📊 Exemplo de uso

Se você tem:

250 VP
Precisa de 1775 VP

O sistema:

Calcula os 1525 VP restantes
Testa combinações de compra
Mostra:
Melhor opção na loja oficial
Melhor opção com cashback
Economia total

Ajudar jogadores a:

Gastar menos dinheiro 💸
